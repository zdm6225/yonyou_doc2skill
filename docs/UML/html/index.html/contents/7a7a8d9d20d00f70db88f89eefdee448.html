ConfluenceToSkillConverter

UMLClass
Yonyou Doc2Skill :: yonyou_doc2skill :: Scrapers :: ConfluenceToSkillConverter

Description

Converts Confluence wiki spaces into AI skill format. Supports both API mode (REST API with spacekey) and export mode (HTML/XML export). Parses page trees, macros, tables, and code blocks. Source: confluencescraper.py
