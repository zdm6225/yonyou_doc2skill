SourceManager

UMLClass
Yonyou Doc2Skill :: yonyou_doc2skill :: MCP :: SourceManager

Description

Manages registry of custom config sources (git repositories) at ~/.yonyou-doc2skill/sources.json. Supports add, get, list, remove, update operations with atomic file writes.
