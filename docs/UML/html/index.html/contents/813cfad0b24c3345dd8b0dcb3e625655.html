SkillSeekerMCPServer

UMLClass
Yonyou Doc2Skill :: yonyou_doc2skill :: MCP :: SkillSeekerMCPServer

Description

FastMCP-based MCP server providing 34 tools for generating AI skills from documentation. Supports stdio and HTTP transports. Tools organized in 7 categories: Config (3), Scraping (11), Packaging (4), Splitting (2), Source (5), Vector DB (4), Workflow (5).
