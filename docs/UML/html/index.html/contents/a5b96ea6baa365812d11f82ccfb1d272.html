Description

Strategy+Factory pattern for multi-platform output. SkillAdaptor ABC defines the interface; 20+ implementations format skills for Claude, Gemini, OpenAI, MiniMax, OpenCode, Kimi, DeepSeek, Qwen, OpenRouter, Together, Fireworks, Markdown, and RAG frameworks (LangChain, LlamaIndex, Haystack, Chroma, FAISS, Qdrant, Weaviate, Pinecone).
