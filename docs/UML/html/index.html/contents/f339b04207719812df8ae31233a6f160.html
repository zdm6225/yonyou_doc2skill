LocalSkillEnhancer

UMLClass
Yonyou Doc2Skill :: yonyou_doc2skill :: Enhancement :: LocalSkillEnhancer

Description

Enhances SKILL.md using local CLI coding agents (Claude Code, Codex, Copilot, OpenCode). No API key needed. Supports headless, background, daemon, and terminal modes. Includes smart summarization for large skills (>30K chars).
