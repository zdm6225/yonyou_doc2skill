UMLPackage
Yonyou Doc2Skill :: yonyou_doc2skill :: Sync

Description

Documentation change detection and sync. Monitors documentation sources for changes using content hashing, HTTP headers (ETag, Last-Modified), and content diffing. Triggers re-scraping when changes are detected and sends notifications.
