Yonyou Doc2Skill :: yonyou_doc2skill :: CLICore

Description

Git-style CLI dispatcher. Entry point for all yonyou-doc2skill commands. Maps subcommands to modules via COMMAND_MODULES registry, handles argument parsing, and delegates to source-specific scrapers via auto-detection.
