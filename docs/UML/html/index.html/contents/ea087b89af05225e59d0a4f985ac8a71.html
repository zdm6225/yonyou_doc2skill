ChangeDetector

UMLClass
Yonyou Doc2Skill :: yonyou_doc2skill :: Sync :: ChangeDetector

Description

Detects changes in documentation pages. Uses multiple strategies: content hashing (SHA-256), Last-Modified headers, ETag headers, and content diffing.
