Enhancement

UMLPackage
Yonyou Doc2Skill :: yonyou_doc2skill :: Enhancement

Description

AI-powered skill enhancement. Supports API mode (direct Claude API) and LOCAL mode (Claude Code CLI). Two enhancement hierarchies: AIEnhancer for API-based enhancement and UnifiedEnhancer for C3.x pipeline output. Controls enhancement depth via --enhance-level 0-3.
