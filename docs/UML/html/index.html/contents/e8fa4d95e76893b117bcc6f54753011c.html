AsciiDocToSkillConverter

UMLClass
Yonyou Doc2Skill :: yonyou_doc2skill :: Scrapers :: AsciiDocToSkillConverter

Description

Converts AsciiDoc files into AI skill format. Discovers .adoc/.asciidoc files, resolves attributes/includes, parses sections, extracts code blocks/tables/admonitions, and generates categorized SKILL.md output. Source: asciidoc_scraper.py
