InstallAgent

UMLClass
Yonyou Doc2Skill :: yonyou_doc2skill :: Packaging :: InstallAgent

Description

Installs skills to AI coding agent directories. Supports 17+ agents (Claude Code, Cursor, VS Code, Amp, Goose, etc.) by copying skill directories to agent-specific installation paths. Handles both global (~/) and project-relative paths with fuzzy name matching.
