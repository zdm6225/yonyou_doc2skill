VideoToSkillConverter

UMLClass
Yonyou Doc2Skill :: yonyou_doc2skill :: Scrapers :: VideoToSkillConverter

Description

Convert video content to Claude skill. Supports YouTube videos/playlists, Vimeo, and local files. Extracts transcripts, metadata, and optional visual content via Whisper and OpenCV.
