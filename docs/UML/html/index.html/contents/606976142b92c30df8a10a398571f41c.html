ManPageToSkillConverter

UMLClass
Yonyou Doc2Skill :: yonyou_doc2skill :: Scrapers :: ManPageToSkillConverter

Description

Converts Unix man pages into AI skill format. Extracts via man command or reads .1-.8 files from directories. Parses sections, options, examples, and see-also references. Strips troff formatting. Source: man_scraper.py
