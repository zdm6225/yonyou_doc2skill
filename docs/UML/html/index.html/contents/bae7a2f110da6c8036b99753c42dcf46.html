ConfigManager

UMLClass
Yonyou Doc2Skill :: yonyou_doc2skill :: Utilities :: ConfigManager

Description

Manages persistent configuration for GitHub profiles, API keys, and scraping settings. Stores config in ~/.config/yonyou-doc2skill/config.json. Supports multiple GitHub token profiles with rotation.
