EpubToSkillConverter

UMLClass
Yonyou Doc2Skill :: yonyou_doc2skill :: Scrapers :: EpubToSkillConverter

Description

Convert EPUB e-book to Claude skill. Uses ebooklib for EPUB parsing, BeautifulSoup for XHTML content extraction. Supports DRM detection and spine-order iteration.
